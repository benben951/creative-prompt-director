# Creative Prompt Director

一个中文优先、默认离线的 AI 视觉提示词导演 Skill。它把创意、剧本、产品资料或参考图整理成结构化、可执行、可验收的中英文提示词。

适合配合即梦、可灵、海螺、Seedance、Veo、Sora、Midjourney、Flux、GPT Image、Gemini 等图片和视频工具使用，但本 Skill 默认不会直接调用这些平台。

## 主要能力

- AI 短剧、广告片、分镜和图生视频提示词
- 人物、场景、服装、道具和镜头连续性控制
- 产品白底图、Hero 图、生活方式图、细节图和组图
- 产品外形、材质、颜色、Logo 和标签保真控制
- 提示词优化、质量评分和失败原因检查
- 隐私、版权/IP、真实人物授权和广告宣传真实性检查

## 使用场景

### 1. AI 短剧

适用于 15～90 秒竖屏短剧、悬疑反转、情感剧情、爽剧、对话戏、动作戏、小说转分镜以及连续剧分集制作。

```text
使用 $creative-prompt-director，把下面的故事改成60秒竖屏短剧。

要求：
- 前3秒出现冲突
- 9:16竖屏
- 现实主义风格
- 女主外貌和服装保持一致
- 结尾留下反转
```

输出包括剧情节拍、时间分配、人物和场景设定、分镜表、镜头时长、生成提示词、台词音效、转场建议和连续性检查。

### 2. 产品广告短片

适用于产品功能广告、品牌情绪广告、开箱、使用演示、电商短视频和产品植入短剧。

```text
使用 $creative-prompt-director，为一款智能保温杯设计30秒产品广告。

平台：抖音
比例：9:16
受众：办公室年轻人
核心卖点：温度显示和长效保温
要求：产品颜色、按钮和屏幕位置不能变化
```

### 3. 电商产品图

适用于白底主图、商品详情图、包装展示、材质特写、尺寸参照图、电商组图和多角度商品图。

```text
使用 $creative-prompt-director，为这款黑色智能保温杯制作电商白底主图提示词。

要求：
- 1:1
- 纯白背景
- 保留杯身形状、按钮和显示屏
- 不增加不存在的配件
- 标签文字后期合成
```

### 4. 产品场景图

适用于家居、户外、美妆、食品饮料、科技产品 Hero 图和社交媒体宣传图。

```text
使用 $creative-prompt-director，为这款护肤精华设计三张生活方式图：

1. 浴室晨间场景
2. 梳妆台产品特写
3. 带留白的品牌Hero图

保持瓶身比例、玻璃材质、液体颜色和标签位置一致。
```

### 5. 提示词优化与审核

```text
使用 $creative-prompt-director，审核并优化下面的产品图提示词。

重点检查：
- 产品是否容易变形
- Logo和包装是否可能出错
- 是否包含虚假宣传
- 是否缺少构图和灯光信息

原提示词：
……
```

Skill 会从目标清晰度、视觉可执行性、约束完整性、连续性或保真、模型可行性、安全真实性六个维度评分。

## 推荐输入格式

资料越明确，结果越稳定：

```text
目标：
生成类型：
目标平台或模型：
比例：
时长或图片数量：
目标受众：
必须保留：
禁止出现：
参考素材：
```

信息不完整时，Skill 最多询问三个高影响问题，其余部分使用明确标注的合理假设。

## 标准工作流程

1. 提取目标、平台、比例、时长和参考素材。
2. 建立创意约定以及必须保持不变的事实。
3. 为短剧拆解剧情节拍、连续性 Bible 和镜头卡；为产品图建立产品保真档案和拍摄方案。
4. 输出中文制作说明和英文可复制提示词。
5. 添加负面约束、连续性或产品保真锁定。
6. 按 12 分制质量门禁检查，并在必要时提供修正版。

## 默认输出

1. `制作说明（中文）`
2. `可复制提示词（英文）`
3. `负面约束`
4. `连续性/保真锁定`
5. `质量评分和风险提示`
6. 必要时提供修正版

## 安装

将本仓库安装到支持 Skills 的 AI 客户端，并保持目录名为 `creative-prompt-director`。不同客户端的 Skills 目录位置不同，优先使用客户端官方安装方式。

安装后可测试：

```text
使用 $creative-prompt-director，为一款黑色智能保温杯设计白底主图提示词。
```

## 安全边界

本仓库只包含 Markdown 和 UI 元数据：

- 不调用图片或视频 API
- 不读取或保存 API Key
- 不自动上传剧本、产品图、人物参考图或品牌素材
- 不安装第三方依赖，不执行 Shell/Python 生成器
- 不编造产品参数、认证、功效、价格或代言信息
- 不保证生成模型能够准确绘制 Logo 和小字

当包装文字或 Logo 必须完全准确时，应生成干净画面后使用原始品牌文件进行后期合成。
